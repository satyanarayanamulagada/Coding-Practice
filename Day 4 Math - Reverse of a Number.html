Problem statement

Write a program to reverse a given number.

Explanation

To reverse a given number, we can use the following steps:

  1. Initialize a variable to store the reversed number (let's call it reversedNum) and set it to 0.
  1. Extract the last digit from the given number using the modulo operator (%). Let's call this last digit digit.
  1. Add digit to reversedNum multiplied by 10.
  1. Remove the last digit from the given number using integer division (/).
  1. Repeat steps 2-4 until the given number becomes 0.
  1. The final value of reversedNum will be the reversed number.

By following these steps, we can reverse a given number.

For example, let's say we want to reverse the number 12345:

  1. Initialize reversedNum to 0.
  1. Extract the last digit, which is 5. Add 5 to reversedNum multiplied by 10 (0 * 10 + 5 = 5).
  1. Remove the last digit from 12345, resulting in 1234.
  1. Repeat steps 2-3.
    • Extract the last digit, which is 4. Add 4 to reversedNum multiplied by 10 (5 * 10 + 4 = 54).
    • Remove the last digit from 1234, resulting in 123.
  1. Repeat steps 2-3.
    • Extract the last digit, which is 3. Add 3 to reversedNum multiplied by 10 (54 * 10 + 3 = 543).
    • Remove the last digit from 123, resulting in 12.
  1. Repeat steps 2-3.
    • Extract the last digit, which is 2. Add 2 to reversedNum multiplied by 10 (543 * 10 + 2 = 5432).
    • Remove the last digit from 12, resulting in 1.
  1. Repeat steps 2-3.
    • Extract the last digit, which is 1. Add 1 to reversedNum multiplied by 10 (5432 * 10 + 1 = 54321).
    • Remove the last digit from 1, resulting in 0.
  1. The final value of reversedNum is 54321, which is the reverse of the given number 12345.

By following this algorithm, we can reverse any given number.

Code Solution

  • ReverseNumber.Java
    public class ReverseNumber {
    public static void main(String[] args) {
        int number = 12345;
        int reversedNum = 0;

        while (number != 0) {
            int digit = number % 10;
            reversedNum = reversedNum * 10 + digit;
            number /= 10;
        }

        System.out.println("The reversed number is: " + reversedNum);
    }
}
