Day 7: Java program to display even numbers from 1 to N

Problem statement

Problem statement

Write a Java program to display even numbers from 1 to N, where N is a positive integer.

Test Cases

Here is a test case for displaying even numbers from 1 to N:

  • Test Case

    Input:
    N = 10

    Output:
    2 4 6 8 10

Explanation

To display even numbers from 1 to N, we can follow these steps:

  1. Take a positive integer N as input.
  1. Iterate through numbers from 1 to N.
  1. Check if each number is divisible by 2 (i.e., an even number).
  1. If a number is even, display it.

For example, if N is 10, the program should display the even numbers 2, 4, 6, 8, and 10.

Code Solution

  • EvenNumbers.java
    public class EvenNumbers {
    public static void main(String[] args) {
        int N = 10;

        for (int i = 1; i <= N; i++) {
            if (i % 2 == 0) {
                System.out.print(i + " ");
            }
        }
    }
}
