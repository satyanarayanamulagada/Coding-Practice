Day 1: Math - Even Or Odd

Problem statement

Is the given number even or odd?

Test Cases

  • Test Case1

    Input: 15

    Output: Odd

  • Test Case2

    Input: 10

    Output: Even

Explanation

To determine if a given number is even or odd, we can use the modulo operator. If the number modulo 2 equals 0, then it is even; otherwise, it is odd.

Code Solution

  • EvenOrOdd.java

    public class EvenOrOdd {
    public static void main(String[] args) {
    System.out.println(isEvenOrOdd(7)); // Output: Odd
    System.out.println(isEvenOrOdd(10)); // Output: Even
    }

    public static String isEvenOrOdd(int number) {
    if (number % 2 == 0) {
    return "Even";
    } else {
    return "Odd";
    }
    }
    }
