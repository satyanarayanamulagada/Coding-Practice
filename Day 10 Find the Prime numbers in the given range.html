Day 10: Find the Prime numbers in the given range

Problem statement

Write a program to Find the Prime numbers in the given range

Test Cases

Here is a test case for finding the Prime numbers in the given range:

  • Test Case1

    Input:
    Number: 10

    Output: 2,3,5,7

  • Test Case2

    Input:
    Number: 20

    Output: 2,3,5,7,11,13,17,19

Explanation

The explanation for finding prime numbers in the given range is as follows:

  1. A prime number is defined as a number greater than 1 that has no positive divisors other than 1 and itself.
  1. To check if a number is prime, follow these steps:
    • Initialize a variable isPrime to true.
    • Iterate from 2 to the square root of the number (inclusive).
    • For each iteration, check if the number is divisible by the current iteration value. If it is, set isPrime to false and break out of the loop.
    • After the loop, check the value of isPrime. If it is still true, the number is prime. Otherwise, it is not.

For example, let's say we want to check if the number 12321 is prime:

  1. Initialize isPrime to true.
  1. Iterate from 2 to the square root of 12321 (inclusive).
  1. For each iteration, check if 12321 is divisible by the current iteration value. Since it is not divisible by any value, isPrime remains true.
  1. Check the value of isPrime, which is true. Therefore, the number 12321 is prime.

Code Solution

  • PrimeFinder.Java
    public class PrimeFinder {
    public static void main(String[] args) {
        int start = 1;
        int end = 20;

        System.out.print("Prime numbers between " + start + " and " + end + ": ");

        for (int i = start; i <= end; i++) {
            boolean isPrime = true;

            for (int j = 2; j <= Math.sqrt(i); j++) {
                if (i % j == 0) {
                    isPrime = false;
                    break;
                }
            }

            if (isPrime) {
                System.out.print(i + " ");
            }
        }
    }
}
