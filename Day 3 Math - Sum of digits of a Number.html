Day 3: Math - Sum of digits of a Number

Problem statement

Find the sum of all digits of a given Number.

Test Cases

Here is a test case for finding the sum of all digits:

  • Test Case3

    Input: 12345

    Output: 15

  • Test Case2

    Input: 121

    Output: 4

Explanation

The given code is not related to finding the sum of all digits of a number. It is a code snippet to determine if a given number is even or odd.

To find the sum of all digits of a number, you can follow these steps:

  1. Initialize a variable sum to 0.
  1. Extract each digit from the given number one by one.
  1. Add each digit to the sum variable.
  1. Repeat steps 2 and 3 until all digits have been processed.
  1. Return the value of sum as the sum of all digits.

Code Solution

  • SumOfDigits.java

    public class SumOfDigits {
    public static void main(String[] args) {
    int number = 12345;
    int sum = findSumOfDigits(number);
    System.out.println("Sum of digits: " + sum); // Output: Sum of digits: 15
    }

    public static int findSumOfDigits(int number) {
    int sum = 0;
    while (number > 0) {
    int digit = number % 10;
    sum += digit;
    number /= 10;
    }
    return sum;
    }
    }
