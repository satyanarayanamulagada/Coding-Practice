Day 13: Java program to check if a given number is perfect square or not.

Problem statement

Write a Java program to check if a given number is a perfect square or not.

Test Case

Here is a test case for checking if a given number is a perfect square or not:

  • Test Case 1

    Input:
    Number = 16

    Output:
    The number 16 is a perfect square.

  • Test Case 2

    Input:
    Number = 25

    Output:
    The number 25 is a perfect square.

  • Test Case 3

    Input:
    Number = 10

    Output:
    The number 10 is not a perfect square.

Explanation

To check if a given number is a perfect square or not, we can follow these steps:

  1. Take a positive integer, number, as input.
  1. Calculate the square root of the number.
  1. If the square root is an integer, then the number is a perfect square. Otherwise, it is not a perfect square.
  1. Display whether the number is a perfect square or not.

For example, if the number is 16, the program should display "The number 16 is a perfect square."

Code Solution

  • PerfectSquare.java
    public class PerfectSquare {
    public static void main(String[] args) {
        int number = 16;
        double squareRoot = Math.sqrt(number);

        if (squareRoot == Math.floor(squareRoot)) {
            System.out.println("The number " + number + " is a perfect square.");
        } else {
            System.out.println("The number " + number + " is not a perfect square.");
        }
    }
}
